#hashtag hub

The hashtag hub is a web service to get links to #hashtag pages on different web platforms. The main purpose of this tool is to be used on Wikidata as the 'formatter URL' (P1630) value of 'hashtag' (P2572), without having to systematically link to a unique centralized platform, hashtags being, most of the time, not bound to a single platform.
